自製函式庫 - 輸入限制

Sample 1 (.myfun-input1)

需求

  • 只能輸入數字。
  • 最小為 0。

範例

Code  

<input type="number" class="myfun-input1" value="0" placeholder="最小為 0" min="0">

設計說明

  • 只能輸入數字，基本上 type 使用 number 即可達成。
  • 給予一個預設值 0 (value)，以及預設說明文字 (placehoder)。
  • min 屬性設定為 0 可以限制遞減箭頭不會小於 0，但無法阻止手動輸入。
  • 監聽 keypress 禁止輸入 - 號。
  • 監聽 paste 禁止貼上不合規格的內容。
  • 聚焦輸入框時，若內容為 0 則自動清空以方便輸入。
  • 離開輸入框時，若內容為空則自動補為 0。
