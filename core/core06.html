提供開發者語法嚴格、受限的模式（strict mode），會影響語法的使用。

this 與箭頭函式

箭頭函式沒有自己的 this，箭頭函式的 this 會指向外層作用域的 this。

this 應用：模擬 Vue 物件操作

物件裡使用 this 的好處：不需要直接指定物件名稱。

當物件裡的屬性之間呼叫都使用 this 時，我們可以輕易地將這個物件複製成另一個物件，並只修改必要的屬性即可使用，而不需要再重寫一遍物件。

原型鏈

「原型鏈」就是可以透過 prototype 一層一層往上找。

可以在中間任何一層 prototype 插入新的方法，其實體物件即可使用這些方法。

好處：

  1. 省記憶體。
  2. 好管理。

JavaScript 本身就是透過原型鏈在運作。
