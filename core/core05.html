函式基本介紹

  • 函式本身由 function 關鍵字定義。
  • 用途：
    1. 可以執行片段程式碼。
    2. 可以重複呼叫。
    3. 可以傳入不同的參數，改變執行結果。
  • 是一個物件（因此可以加入屬性）。
  • 通常使用小寫開頭（大寫函式代表的意義不同）。
  • 限制作用域。
  • 函式可以回傳值。
  • 呼叫函式時會回傳一個值，所以「呼叫函式」這個動作是一個表達式（expression）。

立即函式

  1. 限制作用域。
  2. 立即執行。
  3. 可以回傳結果。
  4. 為函式表達式。

一級函式（First-class Function）

  • 是一個概念，並不局限於 JavaScript。
  • 定義：當函式在那個語言中可以被視為跟其他的變數一樣時，我們稱那樣的程式語言擁有一級函式，例：
    • 可以被當作值一樣指派到另一個變數，例：函式表達式。
    • 函式可以作為參數傳遞到另一個函式（callback function）。
    • 可以被另一個函式作為回傳值。

高階函式

是一個概念，更像是一種技法，並不局限於 JavaScript。

當函式可以接受另一個函式作為參數、或者返回一個函式作為結果時，此函式即為高階函式。（其概念與一級函式有部份重複）
